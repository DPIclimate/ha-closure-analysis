HTTP GET & POST request using cURL.

Basic HTTP request using CURL.

Most initialisation is done outside of this function. This function just handles the HTTP request and puts the data inside a provided cJSON object. Note this function has a memory leak on a Mac M1 -> (curl_easy_perform()) has 13 leaks, totalling 496 bytes per call.

const char *URL = "https://www.example.com";
// Create some headers (more may be needed depending on application)
struct curl_slist *headers = NULL;
headers = curl_slist_append(headers, "Content-Type: application/json");
// Create a cJSON object to hold the response
cJSON *response = NULL;
CURLcode result = HttpRequest(&response, URL, headers, 0, NULL);
// Free memory
curl_slist_free_all(headers);
cJSON_Delete(response);

Helper function to handle a CURL request response which normally gets written to stdout.

Callback function to save HTTP response data to a character array.

CURL defaults to printing to stdout. This function is added to CURL in the setup options (curl_easy_setupopt()). Memory is allocated and then filled with the response string. This character array is then used to create cJSON objects and other items.

MemoryStruct_TypeDef chunk;
chunk.memory = malloc(1);
chunk.size = 0;
curl_easy_setopt(curl, CURLOPT_WRITEFUNCTION, WriteMemoryCallback);
curl_easy_setopt(curl, CURLOPT_WRITEDATA, (void *)&chunk);
free(chunk.memory) // Don't forget to free allocation

Helper function to handle writing timeseries data to a .csv file.
